# Queries

<CH.Scrollycoding>

### QueryStreamsMsg

_`pub stream_ids:`_ A list of stream IDs to query.  

_`pub creator:`_ A creator's address. Streams created by this address will be returned.
  
_`pub recipient:`_ A recipient's address. Streams that transfer to this address will be returned.   

_`pub start_after:`_ A stream ID used for pagination of results. If specified, the page that begins after this ID will be returned.

``` rust minter.rs focus=3:8
#[derive(Serialize, Deserialize, Clone, Debug, PartialEq, JsonSchema)]
#[serde(rename_all = "snake_case")]
pub struct QueryStreamsMsg {
    pub stream_ids: Option<Vec<Uint64>>,
    pub creator: Option<String>,
    pub recipient: Option<String>,
    pub start_after: Option<Uint64>,
}
```

### QueryTokensMsg

Query a specified list of tokens. Pagination limits results to 50 at a time. To view the next 50, specify the last returned token using _`start_after`_. For more info, check out [the cw721 docs](https://github.com/CosmWasm/cw-nfts/blob/main/packages/cw721/README.md#queries-1).

```rust minter.rs focus=3:6
}
```
