# Station for LocalTerra development

[LocalTerra](../../develop/localterra/install-localterra.mdx) can only be used with the Station Classic Extension. To install this extension, follow the steps below. 

<Admonition type="danger" icon="☢️" title="Station Classic is for development only">

The Station Classic download provided in this guide is for development and testing purposes only. This extension should only be used with LocalTerra. For mainnet or testnet functionality, please use the [Official Station Chrome Extension](./download/station-extension.mdx). 

</Admonition>

## Prerequisites

- Download [Google Chrome](https://www.google.com/chrome/).

## Install

1. Download the Station Classic Chrome Extension for developers by clicking the link below. 

    - download link

2. Extract the extension by clicking on the `.zip` file. 

3. Visit [`chrome://extensions/`](chrome://extensions/) in your Chrome browser.

4. If you already have Station installed, find the Station card inside the extensions page and disable it using the toggle in the bottom-right corner of the card.

![](/img/screens/station-classic/1-toggle.png)

5. Enable the developer mode using the toggle in the top-right corner of the page.

![](/img/screens/station-classic/2-dev.png)

6. Click on Load unpacked.

![](/img/screens/station-classic/3-unpacked.png)


7. Navigate to the folder (likely in your downloads folder from step 1), then click select.

8. Done! It’s now installed. (It may give you a manifest version error that you can disregard).

![](/img/screens/station-classic/4-done.png)


9. You can pin it to make it more accessible in your browser.

![](/img/screens/station-classic/5-pin.png)

10. To switch Networks to LocalTerra, click on the extension icon, then click the gear icon in the upper-right corner of the screen.
