# Switch networks

This guide will walk you through the process of changing networks on Station.

<Admonition type="warning" icon="ðŸ›°" title="Station Classic for development">

Terra Classic can only be accessed using the Terra Station Classic desktop or mobile application. To download this application, visit the [Station Classic desktop repo](https://github.com/terra-money/station-desktop/releases/tag/v1.2.0-legacy) on Github. To install the mobile application, visit the [mobile installation guide](./download/station-mobile.mdx). 

</Admonition>

1. Open Station and connect your wallet.

2. Click the settings icon in the upper right-hand corner. In the Station desktop app, the settings icon is a globe. In the browser extensions, the icon is a gear.

3. Navigate to the **Network** tab and select the network you would like to connect to.

<Admonition type="tip" icon="ðŸ›°" title="Station Classic for development">

LocalTerra can only be used with the Station Classic Developer Extension. To install this extension, follow the [Station Classic for development installation guide](../../learn/station/localterra.mdx). 

</Admonition>
