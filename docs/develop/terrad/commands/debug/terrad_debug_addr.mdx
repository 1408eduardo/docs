# terrad debug addr

Convert an address between hex encoding and bech32.

## Example

```sh Terrad
terrad debug addr terra1e0jnq2sun3dzjh8p2xq95kk0expwmd7shwjpfg
```

## Usage

```sh Terrad
terrad debug addr [address] [flags]
```

## Options

```sh Terrad
  -h, --help   help for addr
```
