# terrad debug raw-bytes

Convert raw bytes output (`[10 21 13 255]`) to hex. 

## Synopsis

Convert raw-bytes to hex.

## Example

```sh Terrad
terrad debug raw-bytes [72 101 108 108 111 44 32 112 108 97 121 103 114 111 117 110 100]
			
```

## Usage

```sh Terrad
terrad debug raw-bytes [raw-bytes] [flags]
```

## Options

```sh Terrad
  -h, --help   help for raw-bytes
```
