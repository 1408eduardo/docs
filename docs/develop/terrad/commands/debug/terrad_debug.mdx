# terrad debug

A tool to help with app debugging.

```sh Terrad
terrad debug [flags]
```

## Options

```sh Terrad
  -h, --help   help for debug
```

## See also

* [terrad debug addr](./terrad_debug_addr.mdx)	 - Convert an address between hex and bech32. 
* [terrad debug pubkey](./terrad_debug_pubkey.mdx)	 - Decode a pubkey from proto JSON. 
* [terrad debug raw-bytes](./terrad_debug_raw-bytes.mdx)	 - Convert raw bytes output (`[10 21 13 255]`) to hex.
