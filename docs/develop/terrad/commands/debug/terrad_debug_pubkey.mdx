# terrad debug pubkey

Decode a pubkey from proto JSON. 

## Synopsis

Decode a pubkey from proto JSON and display its address.

**Example**:

```sh Terrad
$ terrad debug pubkey '{"@type":"/cosmos.crypto.secp256k1.PubKey","key":"AurroA7jvfPd1AadmmOvWM2rJSwipXfRf8yD6pLbA2DJ"}'
			
```

## Usage

```sh Terrad
terrad debug pubkey [pubkey] [flags]
```

## Options

```sh Terrad
  -h, --help   help for pubkey
```
