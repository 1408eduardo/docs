# terrad keys list

List all keys. 

## Synopsis

Return a list of all public keys stored by this key manager along with their associated names and addresses.

## Usage
```sh Terrad
terrad keys list [flags]
```

## Options

```sh Terrad
  -h, --help         help for list
  -n, --list-names   List names only
```
