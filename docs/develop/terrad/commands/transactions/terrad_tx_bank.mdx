# terrad tx bank

Bank transaction subcommands. 

## Usage

<Admonition type="tip">

In this tx, the _`--from`_ flag is ignored, as it is implied from _`[from_key_or_address]`_. When using `--dry-run`, a key name cannot be used, only a `bech32` address.

</Admonition>

```sh Terrad
terrad tx bank [flags]
```
