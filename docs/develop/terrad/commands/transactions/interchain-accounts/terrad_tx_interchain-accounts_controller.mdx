# terrad tx interchain-accounts controller

This command accepts a message string and serializes it into packet data which is outputted to _`stdout`_. It can be used in conjunction with _`send-tx`_ which submits pre-built packet data containing messages to be executed on the host chain.

## Usage

```sh Terrad
terrad tx interchain-accounts controller [command]
```

## Subcommands

```sh Terrad
register    Register an interchain account on the provided connection.
send-tx     Send an interchain account tx on the provided connection.
```
