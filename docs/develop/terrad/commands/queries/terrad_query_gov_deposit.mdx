## terrad query gov deposit

Query details of a deposit

### Synopsis

Query details for a single proposal deposit on a proposal by its identifier.

Example:
$ terrad query gov deposit 1 terra1skjwj5whet0lpe65qaq4rpq03hjxlwd9nf39lk

```sh Terrad
terrad query gov deposit [proposal-id] [depositer-addr] [flags]
```
