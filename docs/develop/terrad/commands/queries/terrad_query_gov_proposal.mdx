## terrad query gov proposal

Query details of a single proposal

### Synopsis

Query details for a proposal. You can find the
proposal-id by running "terrad query gov proposals".

Example:
$ terrad query gov proposal 1

```
terrad query gov proposal [proposal-id] [flags]
```

### Options

