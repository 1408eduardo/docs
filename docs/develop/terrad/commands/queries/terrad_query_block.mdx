# terrad query block

Get verified data for a block at a given height.

## Usage

```sh Terrad
terrad query block [height] [flags]
```

## Options

```sh Terrad
  -h, --help          help for block
  -n, --node string   Node to connect to (default "tcp://localhost:26657")
```
